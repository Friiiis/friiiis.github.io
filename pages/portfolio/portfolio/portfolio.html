Portfolio

A page to display my works and skills.

Case:

Build a beautiful and minimalistic site, to showcase my works, skills and ideas. The portfolio will be used when applying for regular and freelance jobs.

My requirements for the page was, that it should be build with vanilla HTML, CSS and JavaScript, and that it should be responsive and nice to use on any device.

I wanted the page to be unique which is why i decided to make a SPA (single page application). This would probably have been easier with some framework, but since I wanted to make the page purely with vanilla JavaScript, I had to make the SPA structure and animations myself.
